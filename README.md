# 🎨 Course Poster — 课程知识海报生成器

根据任意课程名称，自动生成正反两面 A4 知识海报（正面：知识图谱/概念地图；背面：核心思想金句）。支持用户选择页面风格和主题配色。

## ✨ 功能

- **正面海报**：知识图谱 — 按模块分组的核心概念卡片，每卡片含3-5个关键概念
- **背面海报**：20句核心思想 — 大编号金句卡片，模块色标，视觉层次分明
- **5种页面风格**：学术雅致 / 暗金奢华 / 赛博科技 / 极简素净 / 自然温润
- **6套配色方案**：经典蓝青 / 海洋深蓝 / 森林绿意 / 日落暖调 / 赛博霓虹 / 玫瑰金 + 自定义
- **A4精确适配**：自动溢出检测+字号间距迭代优化，确保打印完美
- **2x高清输出**：1588×2246px PNG，打印清晰锐利

## 📁 结构

```
course-poster/
├── SKILL.md                        # 主文件：触发条件 + 完整工作流
├── references/
│   ├── style-presets.md             # 5种风格 + 6套配色方案
│   ├── front-template.md            # 正面海报模板规范
│   └── back-template.md             # 背面海报模板规范
└── scripts/
    └── screenshot.js                # Playwright HTML→PNG 截图脚本
```

## 🚀 使用方式

在 WorkBuddy 中导入本 skill，然后：

1. 说「帮我做个XX课程的知识海报」→ 触发 skill
2. 选择页面风格和配色
3. 自动生成正反面 HTML + PNG

## 🛠 技术栈

- HTML + CSS（mm/pt单位，打印适配）
- Google Fonts: Noto Sans SC + Noto Serif SC
- Playwright：HTML→PNG 截图 + 溢出检测
- A4尺寸：210×297mm / 794×1123px(96dpi) / 2x输出
