# 风格预设与配色方案

## 页面风格

### 1. 学术雅致（Academic）
- 背景：暖白 `#F7F5F0`
- 卡片：纯白 `#FFFFFF` + 浅灰边框
- 字体：衬线标题 + 无衬线正文
- 装饰：极简线条、低透明度SVG纹理
- 适用：经管、人文、社科类课程

### 2. 赛博科技（Cyber）
- 背景：深蓝黑 `#0A0E1A`
- 卡片：半透明深色 `rgba(255,255,255,0.05)` + 发光边框
- 字体：无衬线，部分英文用等宽
- 装饰：网格线、霓虹发光、扫描线
- 适用：编程、AI、科技类课程

### 3. 极简素净（Minimal）
- 背景：纯白 `#FFFFFF`
- 卡片：无边框，仅色条分隔
- 字体：全部无衬线
- 装饰：无，仅靠字号和色块区分层级
- 适用：设计、艺术、思维类课程

### 4. 自然温润（Nature）
- 背景：米色 `#FAF6F0`
- 卡片：浅米 `#FFF9F0` + 暖棕边框
- 字体：衬线标题 + 圆体正文
- 装饰：叶片/水纹SVG纹理
- 适用：健康、心理、教育类课程

### 5. 暗金奢华（Luxe）
- 背景：深炭 `#1C1C1E`
- 卡片：深灰 `#2C2C2E` + 金色边框
- 字体：衬线标题 + 细体正文
- 装饰：金色线条、菱形纹理
- 适用：金融、商业、高端课程

## 用户自定义配色

用户可提供任意主色，按以下规则生成4色：
1. 主色 = 用户色
2. 辅色1 = 主色色相旋转60°
3. 辅色2 = 主色互补色
4. 辅色3 = 主色色相旋转180°后降饱和度

## 风格×配色组合

所有风格可与所有色板自由组合。暗色风格（Cyber/Luxe）使用色板时需调高亮度以确保暗底可读性。
